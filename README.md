# 海岛奇兵雕刻序列模拟器 (Boom Beach Sculptor Simulator)

这是一个用于模拟海岛奇兵游戏中雕刻机雕刻序列的网页工具。

## 🎮 项目概述

海岛奇兵的雕刻系统允许玩家通过消耗资源来获取不同的雕刻加成，每种雕刻都提供独特的战斗加成。本模拟器帮助玩家：

- 模拟不同等级的雕刻加成
- 优化雕刻序列规划
- 计算资源需求
- 导出/导入配置数据

## ✨ 主要功能

### 🏛️ 雕刻系统
- **12种不同类型的雕刻**，每种都有独特的加成效果
- **4个雕刻槽位**：绿色、蓝色、红色、紫色
- **2种升级**：金币升级、经验升级
- **智能推荐系统**：根据当前序列推荐最优雕刻

### 📊 数据管理
- **自动保存**：所有进度自动保存到本地存储
- **导入/导出**：支持JSON格式的配置文件导入导出
- **历史记录**：完整的雕刻历史记录，包含时间戳

### 🎨 界面特性
- **响应式设计**：适配各种屏幕尺寸
- **可折叠侧边栏**：最大化主界面使用空间
- **实时预览**：实时查看当前雕刻配置效果
- **多主题支持**：深色主题，模拟游戏界面风格

## 🛠️ 技术特点

## 📋 使用说明

### 🚀 快速开始
1. 打开 `SCULPTOR MASTER 海岛奇兵雕刻模拟器.html`
2. 页面会自动加载上次的进度
3. 点击"生成测试数据"可快速生成示例配置

### 📝 基本操作流程
1. **选择雕刻**：在表格中点击绿色、蓝色、红色或紫色格子
2. **选择品质**：在弹窗中选择金币或经验升级
3. **确认配置**：点击"已获取"按钮确认当前配置
4. **序列推进**：系统自动推进到下一序列
5. **查看效果**：在弹窗中查看雕刻加成效果

### 💾 数据管理
- **导出配置**：点击顶部"导出数据"按钮
- **导入配置**：点击"导入数据"并选择JSON文件
- **重置数据**：点击"重置所有记录"
